-----------INSTANCES NAME------------

Standard instance name: PSC*-C**-***

PSC = Plant-Satellite-Customer
* = Instance number (1 to 5)
C = Class
** = Class number (1 to 5)
*** = Number of plants (50 or 100)

------HOW TO READ THE INSTANCES------

1. 1st row is I, J, K (# of plants, satellites and customers, respectively)
2. K rows with customer demand
3. I rows with plant capacity and fixed cost, respectively
4. Matrix I rows per J columns with variable cost from plant to satellite
5. J rows with satellite capacity and fixed cost, respectively
6. Matrix J rows per K columns with variable cost from satellite to customer
